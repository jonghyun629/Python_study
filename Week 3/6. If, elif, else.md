# ğŸ”¹ 2. `if / else`

The `else` block runs if the `if` condition is **False**.

```python
    print("You are an adult")

    print("You are a minor")

```

ğŸ‘‰ Output: `You are a minor`

# ğŸ”¹ 3. `if / elif / else`

`elif` (short for *else if*) checks another condition if the first is `False`.

```python
score = 85

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Grade: F")

```

ğŸ‘‰ Output: `Grade: B`

---

# ğŸ”¹ 4. Nested `if`

You can put an `if` inside another `if`.

```python
age = 20
has_id = True

if age >= 18:
    if has_id:
        print("You can enter.")
    else:
        print("You need an ID.")
else:
    print("You are too young.")

```

# ğŸ”¹ 5. Short `if` (Ternary Operator)

You can write it in one line:

```python

print(result)   # Odd

```
